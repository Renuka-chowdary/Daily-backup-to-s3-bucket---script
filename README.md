# Daily-Backup-script

This script takes daily backup of Gitlab/Jenkins/Sonar data to s3 bucket and sends mail.<br/>

1. create a file<br/>
     
     vi s3-script.sh<br/>
     
   copy and paste the script.<br/>
   
2. Run it.<br/>
      sh s3-script.sh

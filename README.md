# Daily-Backup-script

This script takes daily backup of Gitlab/Jenkins/Sonar data to s3 bucket and sends mail.<br/>

1. create a file
      vi s3-script.sh
   copy and paste the script.
   
2. Run it.
      sh s3-script.sh
